# 《数学分析》课后习题项目

这个项目计划用半年到一年的时间，将中科大《数学分析》第2版所有课后题再做一遍，以复习数分。第2版是比较老的教材版本，里面有一些被后续版本删除的章节，但题目并没有过时。在B站上也有与该版本教材配套的史济怀老师的课堂录像，该录像应录于2004年秋。

![shufen](../images/shufen.jpg)
