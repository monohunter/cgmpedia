# 前言

欢迎来到我的个人wiki笔记本，这里是我的学习记录的分享平台。

在这个AI时代，知识是触手可及的，以前需要多年学习才能获得的知识，似乎问一句就能得到。但此时此刻，想要真正解决问题，经典的基础知识却比以往更加重要。当今比任何时代都更加需要真才实学。

我们可以think big，但真正能让愿景落地，就需要具备硬核(hardcore)基本功，它主要包括两个领域：

1. 数学，它决定了底层算法；
2. 化学，它决定了材料合成。

于是这里就是回归根本，锻炼基本功的地方。
